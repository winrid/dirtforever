// INSTALL DIRTFOREVER

Get connected in a few minutes. You'll run a small local server that redirects the game's network calls to dirtforever.net for clubs, leaderboards, and stage results.

// PREREQUISITES

Windows 10/11 Required for the hosts redirect
Python 3.9+ python.org/downloads
DiRT Rally 2.0 Installed via Steam
cryptography Run: pip install cryptography

// SETUP (ONE TIME)

Create an Account

Register at dirtforever.net if you haven't already.

Generate a Game Token

Log in and go to your Dashboard — click Generate Game Token and copy it. This links the local server to your account.

Download the Server

Clone or download the repository from GitHub.

git clone https://github.com/winrid/dirtforever.git

Run the Installer

Open PowerShell as Administrator and run:

cd path\to\dirtforever powershell -ExecutionPolicy Bypass -File scripts\install.ps1

The installer will generate and trust a TLS certificate, redirect Codemasters hostnames to your machine, ask you to paste your game token, and create a DirtForever Server desktop shortcut. UAC will prompt for administrator access.

Start the Server

Double-click the DirtForever Server shortcut on your desktop, or run manually:

python server.py --ssl-cert runtime\certs\dr2server-cert.pem --ssl-key runtime\certs\dr2server-key.pem

Launch the Game

Start DiRT Rally 2.0 via Steam as usual. Game traffic will reach your local server automatically, which syncs with dirtforever.net for clubs, events, and leaderboards.

// COMPATIBILITY

Working

  • Login
  • Events / Career mode
  • Clubs with custom events
  • Stage time submission & leaderboards
  • Vehicle select, repairs, tuning

Not Yet

  • Full championship progression
  • Multiplayer / Time Trial ghosts
  • In-game currency / store

// UNINSTALLING

To remove the hosts redirect, open PowerShell as Administrator and run:

powershell -ExecutionPolicy Bypass -File scripts\remove_windows_redirect.ps1

The certificate installed during setup can be removed manually via certmgr.msc. Config is stored at %APPDATA%\DirtForever\config.json.
